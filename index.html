HTML Шпаргалка

Метаданные документа

< html >< /html > - корневой элемент html-документа < head >< /head > - контейнер для метаданных html-документа < title >< /title > - заголовок / имя html-документа < link > - подключает внешние таблицы стилей < meta > - мета-данные веб-страницы < style >< /style > - подключает таблицы стилей

Секционные элементы (Семантика)

< body >< /body > - тело html-документа < article >< /article > - раздел контента, образующий независимую часть документа или сайта < section >< /section > - логическая область (раздел) страницы, обычно с заголовком < nav >< /nav > - раздел документа, содержащий навигационные ссылки по сайту < aside >< /aside > - контент страницы, имеющий косвенное отношение к основному контенту < h1 > --- < /h6 > - заголовки шести уровней < header >< /header > - секция для вводной информации сайта или группы навигационных ссылок < footer >< /footer > - секция для нижнего колонтитула документа или раздела

Группировка содержимого

< p >< /p > - параграфы в тексте < address >< /address > - контактные данные автора документа или статьи < hr > - горизонтальная линия < pre > < /pre > - выводит текст с пробелами и переносами < ol >< /ol > - упорядоченный нумерованный список < ul >< /ul > - маркированный список < li >< /li >- элемент списка < dl >< /dl > - контейнер для термина и его описания < dt >< /dt > - задаёт термин < dd > < /dd > - расшифровывает термин < figure >< /figure > - независимый контейнер для такого контента как изображения, диаграммы и т.п. < figcaption >< /figcaption > - заголовок для элемента < figure > < main >< /main > - контейнер для уникального основного содержимого в пределах одной страницы сайта < div >< /div > - контейнер для разделов html-документа, группирует блочные элементы

Табличные данные

< table >< /table > - html-таблица < caption >< /caption > - подпись к таблице < colgroup >< /colgroup > - контейнер для одного или нескольких < col > < col > - выбирает для форматирования столбцы < thead >< /thead > - блок заголовков таблицы < tbody >< /tbody > - тело таблицы < tfoot >< /tfoot > - нижний колонтитул таблицы < tr >< /tr > - строка таблицы < th >< /th > - заголовок столбца таблицы < td > < /td > - ячейка таблицы

Скрипты

< script >< /script > - подключает сценарии к странице < noscript >< /noscript > - секция, не поддерживающая скрипт < template >< /template > - фрагменты HTML-кода, которые могут быть клонированы и вставлены в документ скриптом < canvas >< /canvas > - холст-контейнер для динамического отображения изображений

Встроенное содержимое

< picture >< /picture > - контейнер для одного < img / > и ноль или больше < source > < source > - местоположение и тип альтернативных медиаресурсов для элементов < picture >, < video >, < audio > < img > - html-изображения < iframe > < /iframe > - создаёт встроенный фрейм < embed >< /embed > - встраивает внешний интерактивный контент или плагин < object >< /object > - контейнер для встраивания мультимедиа < param >- задаёт параметры для плагинов, встраиваемых с помощью элемента
